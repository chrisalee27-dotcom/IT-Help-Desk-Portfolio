# TICKET-005: Account Locked Out / Password Reset Request

**Ticket Opened:** Simulated phone ticket  
**Priority:** High (user can't log in)  
**Environment:** Active Directory / Microsoft 365 hybrid

**User Description:**  
"Locked out of my account – forgot password."

**Initial Intake:**  
- Verified identity (security questions / manager confirmation)  
- Self-service portal tried? (No access)

**Troubleshooting Steps:**  
1. Confirmed lockout via AD (simulated).  
2. Unlocked account in AD Users and Computers (or Azure AD portal).  
3. Forced password reset (temporary password sent via manager/email).  
4. Guided user through reset and MFA setup.  
5. Tested login success.

**Resolution:**  
User back in system.  
**Time to Resolve:** ~15 minutes  
**Preventive Note:** Encourage MFA and password manager use.

**Lessons / Interview Talking Point:**  
Password/account issues are 30%+ of Tier 1 tickets. Verification + quick unlock is key for MSP fast response.
