# TICKET-006: Cannot Install Approved Software

**Ticket Opened:** Simulated portal ticket  
**Priority:** Medium (blocks specific task)  
**Environment:** Windows 11, company-approved app (e.g., Adobe Reader)

**User Description:**  
"Installer fails – error 'not compatible' or crashes midway."

**Initial Intake:**  
- Downloaded from official source? (Yes)  
- Run as admin tried? (No)

**Troubleshooting Steps:**  
1. Remotely connected.  
2. Ran installer as Administrator.  
3. Checked compatibility mode (right-click → Properties → Compatibility tab → Win 10).  
4. Disabled antivirus temporarily during install.  
5. Installed successfully.  
6. Verified app opens.

**Resolution:**  
Software installed and working.  
**Time to Resolve:** ~25 minutes  
**Preventive Note:** Always run installers elevated.

**Lessons / Interview Talking Point:**  
Many install failures are permissions/compatibility. Remote fix keeps user productive.
